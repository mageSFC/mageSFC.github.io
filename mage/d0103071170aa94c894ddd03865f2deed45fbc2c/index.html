gerrit学习之gerrit源码学习之提交页面

发表于2022-05-30|更新于2022-05-31|gerrit
|阅读量:

gerrit-3.4.3 的编译

对于 一个 提交 页面， 整个 页面是 在 mainContent 下面的

  1. div id=”mainContent” > polygerrit-ui\app\elements\change\gr-change-view\gr-change-view_html.ts

  2. 最上部分的 changeInfo 块。在 mainContent 里面
  3. 在这个文件中 polygerrit-ui\app\elements\change\gr-change-view\gr-change-view_html.ts

  4. moreActions 最右边 3个 点的那个 按钮.

需求： 修改 页面上 commit id 显示成完整的，默认是只显示前几位的。

操作方法： 先浏览器上 右键 选择 检查 查看 这个html元素， 是在哪里，看在哪个div里面？ 有什么 关键词？ id 什么的？
这里实际 搜到了 一个 gr-commit-info 关键词。然后还看出来这个 commit id是在 一个 标签中。
右边还有个 复制到剪切板的 按钮。这个按钮有个提示 Copy full SHA to clipboard 这个也是个关键词。
我们就拿 gr-commit-info关键词 去代码里面搜，如果没搜到 再拿 Copy full SHA to clipboard 去搜。

总之 这一步就是要找到 我们要修改的 地方 在源码的 哪里的。。。。

最终我们搜到 这个文件 polygerrit-ui\app\elements\change\gr-commit-info\gr-commit-info.ts, 在render() 方法中我们看到了想要的 标签，和旁边的 复制到剪切板
